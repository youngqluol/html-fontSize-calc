# html-fontSize-calc

一个用于计算 html fontSize 的工具，用于 rem 适配方案。

## 安装 

```bash
npm install html-fontSize-calc
```

## 使用

```js
// ESM
import { calc } from 'html-fontSize-calc';

calc(750); // 750 是设计稿的宽度
```

```js
// CommonJS
const { calc } = require('html-fontSize-calc');

calc(750); // 750 是设计稿的宽度
```
